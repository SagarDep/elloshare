ElloShare is a photo sharing app for Android. It integrates with Gallery to make it easy to post images from your phone into Ello. Use is simple. Just install it from the Play Store and then use it either from the Launcher or from the sharing button in Gallery.

Implementation is currently somewhay rough. Please fork and modify. Ello is about sharing art, and for many of us, that's mobile pictures.

None of this would have been possible without the Ello research done by github user conatus.
